### Mac

C –> Ctrl

S –> Shift

M –> Alt

Cmd –> Command

| 功能                                    | 快捷键           |
| :-------------------------------------- | :--------------- |
| 自动完成                                | C-Space          |
| 显示/隐藏侧边栏                         | Cmd-0            |
| 切换已打开的文件(GotoPreviousInHistory) | M-Tab            |
| 上/下一行                               | C-p/C-n          |
| 前进/后退一个字符                       | C-f/C-b          |
| 删除一个单词                            | M-Del            |
| 构建                                    | Cmd-b            |
| 运行                                    | Cmd-r            |
| 调试                                    | Cmd-y            |
| 注释                                    | Cmd-/            |
| 换行                                    | Cmd-Return       |
| 跳到定义                                | F2               |
| 切换头文件与源文件                      | F4               |
| 前进/后退                               | M-Cmd-Left/Right |
| 打开定位器                              | Cmd-k            |
